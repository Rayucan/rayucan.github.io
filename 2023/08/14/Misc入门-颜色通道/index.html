misc53

LSB 隐写

misc54

同上，在 Alpha Plane 0 Green Plane 0 Blue Plane 0 发现了隐写痕迹

不同的是这次是竖向排列

勾选对应选项，更改颜色通道读取顺序，得到 flag

misc55

打开图片发现是倒着的，写脚本摆正

from PIL import Image
img=Image.open("misc55.png")
img.transpose(Image.Transpose.FLIP_TOP_BOTTOM).save("output.png")

StegSolve 打开，发现 R0 G0 B0 有隐写痕迹，竖向排列，勾选对应选项

文件头是 zip 格式，Save Bin 保存，解压得到 flag
